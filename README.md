# attendance_data
Attendance Pattern Analysis is a minor data analysis project that focuses on analyzing student attendance records to identify irregular attendance patterns
The analysis helps educational institutions identify students at risk due to low attendance and supports data-driven decision-making. Based on the analysis, suitable improvement measures are suggested to enhance student participation and academic performance.
Objectives

Analyze student attendance data

Identify irregular attendance patterns

Perform frequency analysis

Visualize attendance trends

Suggest improvement measures
Tools & Technologies

Python

Pandas

Matplotlib

Jupyter Notebook

Methodology

Create or load attendance dataset

Calculate attendance percentage

Classify students using a 75% threshold

Perform frequency analysis
Output

Attendance percentage table

Regular vs Irregular frequency count

Attendance visualization graph

Suggested improvement measures
